# correios-cli

A CLI utility to track objects from Correios SRO.

![correios-cli example](correios-cli-example.gif)

## Installation

```
npm install -g correios-cli
```

> Note: If you are using Unix based systems, remember to add `sudo` on the command above.

## How to use

After installation, you can open your Terminal emulator and use the `correios-cli` package globally.

Type `correios-cli track AA123456789BR` to get information about an object.
